Recruiter Demo

vLLM-Powered Chatbot on AWS

Recruiter demo: a simple production-style GenAI chatbot using vLLM on AWS. This demo shows how a static frontend connects to AWS API Gateway, a Flask backend on EC2, and a local vLLM inference server running an open-source instruct model.

Architecture

GitHub Pages frontend
AWS API Gateway HTTP API
Flask backend on EC2 GPU
Local vLLM inference server
Open-source instruct model

Why vLLM

Self-hosted inference
OpenAI-compatible API
Open-source model serving
Separation of backend logic and inference

Tech Stack

Frontend: GitHub Pages
API: AWS API Gateway
Backend: Flask (Python)
Inference: vLLM
Model: open-source instruct model

Request Flow

User → Frontend → API Gateway → Backend → vLLM → Model → Response
